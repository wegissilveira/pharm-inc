### Para rodar o projeto:
```bash
# Clone o repositório
git clone https://github.com/wegissilveira/pharma-inc.git

# Acesse o diretório
cd dir_do_projeto

# Instale as dependencias
npm install

# Inicie o projeto
npm start
```
